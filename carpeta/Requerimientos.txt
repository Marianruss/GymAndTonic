# Requerimientos

- Sistema
    
    El sistema debe permitir el registro de nuevos usuarios.
    
    El sistema debe segmentar las actividades que realiza el gimnasio. Principalmente, va a estar segmentado por dos partes: Musculación y Recreativo.
    
    El sistema debe mostrar una rutina de ejercicios a los usuarios que estén registrados a actividades de Musculación.
    
    El sistema debe diferenciar a los usuarios, siendo estos: Coach, Cliente y Administrador.
    
    El sistema debe permitir que el Coach pueda manipular las rutinas de ejercicios de los clientes de Musculación.
    
    El sistema debe permitir el cobro de suscripciones a los usuarios.
    
    El sistema debe permitir que el cliente pueda modificar su suscripción.
    
    El sistema debe permitir diferenciar entre las sucursales de gimnasio que ofrece la compañía.
    
    El sistema debe listar las distintas actividades Recreativas.
    
    En las actividades Recreativas, el sistema debe permitir al usuario agendar turnos en su actividad deseada.
    
    El sistema debe diferenciar entre dos suscripciones: Plan Local y Plan Total. 
    
    En caso de plan Local, el sistema debe tomar en cuenta los siguientes criterios:
    
    - El cliente sólo puede asistir a una sucursal.
    - El cliente sólo puede agendar turnos Recreativos de esa misma sucursal.
    
    En caso de plan Global, el sistema debe tomar en cuenta los siguientes criterios:
    
    - El sistema debe tener un módulo de turnos, en los cuáles esto consiste en:
        - Un profesor/coach asignado.
        - Una actividad Recreativa.
        - Una sucursal.
        - Un listado limitado de Clientes/Alumnos.
    
    El sistema debe poder utilizarse tanto en la computadora como en el teléfono celular.
    
- Usuario
    
    Los usuarios deben poder sacar turno de su actividad que quiera hacer en el gimnasio.
    
    En los casos de que el cliente modifique su suscripción, el cobro debe aplicarse al abono del próximo mes.
    
    - El cliente puede asistir a cualquier sucursal de la empresa.
    - El cliente puede agendar turnos Recreativos de cualquier sucursal de la empresa.
